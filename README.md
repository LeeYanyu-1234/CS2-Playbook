<h1><center>CS2-Playbook</center></h1>

A personal playbook for CS2, it contains the utilities I know, strategies I use, and notes I write.



## 1 前言

博主是一个24年9月才入坑CS2的小菜鸟，之前也没有FPS基础（小学玩过生死狙击算吗？），目前完美 B、5e 1800分（截止2025年8月5日）。创作本战术纸的初衷也只是整理一下自己作为一个新手学到的各种CS2知识，方便自己查阅。受限于个人水平，本战术纸一定存在各种错误和漏洞，还望各位多多包涵，也欢迎在 Issue 讨论。

由于博主自身的游戏时长并不长，所以战术纸中的大部分内容都是来自于互联网收集，博主会尽量在文末注明出处，但是难免有所遗漏，如果侵犯了原作者的权益，请随时联系博主，博主会及时对内容进行注明或删除。

本来是打算在B站发布专栏的，但是B站的编辑器用起来实在是不方便，博主还是更喜欢用 Typora 编辑，已经在B站发布的篇章除了开篇之外都会陆续进行删除，不过一些视频还是会继续发布在B站的。



## 2 概述

在战术纸开篇之处，博主想先问大家一个问题：你希望打什么样的CS？是个人英雄主义式的CS，能够一个人大杀四方，主导战局；或是团队协作式的CS，队员之间积极沟通、执行战术，用协同弥补个人能力的不足；亦或是娱乐化的CS，用更好的地图理解戏弄对手当老六，把电击枪作为自己的主战武器。不论你喜欢什么样的CS，请明确自己的目标，然后带着目标来进行练习。

这里博主也分享一下自己的想法，博主希望能够打出优美的团队CS，队员的枪法不需要太硬，段位也不需要太高，只要能够每张地图准备几套既定战术，在场上每个人有麦，会及时反馈自己收集到的信息，便于做出决策，然后需要的道具不给呲，不给重，最后坚决执行战术就可以了。遗憾的是说起来容易做起来难，CNCS的环境就决定了路人局能打出的战术其实很少，而凑齐一个较为固定的5排车队在现在来说又是十分困难的，所以大部分情况下枪法都比战术更加重要。
